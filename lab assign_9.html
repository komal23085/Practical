3. Document Object Model (DOM)

The DOM is a programming interface for web documents. It represents the page so that programs (like JavaScript) can change the document structure, style, and content. The DOM represents the document as a tree of objects.

Selecting & Modifying Elements

You can select elements using methods like document.getElementById() or document.querySelector().
