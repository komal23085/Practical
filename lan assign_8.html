JavaScript Control Flow

Control flow is the order in which the computer executes statements in a script.

1. Conditional Statements

  • if...else: Executes a block of code if a condition is true.
  • switch: Selects one of many code blocks to be executed based on a variable's value.

2. Loop Comparison

Loops are used to run the same code over and over again, each time with a different value.

Loop Type Best Used When... Condition Check
for You know exactly how many times you want to loop (e.g., through an array). Before each iteration.
while You don't know the count, but you have a condition that must be true to start. Before each iteration.
do...while The code must run at least once, regardless of the condition. After each iteration.
