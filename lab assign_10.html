JavaScript Array Methods

In JavaScript, arrays are used to store multiple values in a single variable. Here are four fundamental methods used to add or remove elements:

1. push()

Adds one or more elements to the end of an array and returns the new length.

2. pop()

Removes the last element from an array and returns that element.

3. shift()

Removes the first element from an array and returns that element. This shifts all subsequent elements to a lower index.

4. unshift()

Adds one or more elements to the beginning of an array and returns the new length.
